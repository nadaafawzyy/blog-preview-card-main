# Blog Preview Card

A responsive **Blog Preview Card Component** built from the *Frontend Mentor* challenge.  
This project focuses on creating a card UI that displays a featured blog post with image, title, description, author, and read link — using only HTML and CSS.

## 📌 Overview

This repository contains a blog preview card with:

- Feature image  
- Blog title and description  
- Author info  
- “Read More” button  
- Clean, modern, responsive design

This component adapts to different screen sizes and highlights content hierarchy in a visually appealing layout.

## 🧠 Built With

- HTML5  
- CSS3  
- Responsive design techniques
